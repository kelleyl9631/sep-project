Code to Success

In 9th grade, we start off with a program called Scratch. It is a program in which we pseudocode, or make things happen in human reading language rather than computer language.

We also studied robotics, using LEGO MINDSTORM robots. We were able to control the robots, made the robot perform certain actions and created games.



In 10th grade, we learned about processing, data, and networking. For data, we learned to collect information and turn them into spreadsheets and infographics.

We also learned about networking. Networks were about the different branches and to help us understand computers and how the internet works.



In 11th grade, we learned more about web designing and made our own websites. We futhered our skills about websites using CSS3, jQuery, Bootstrap and HTML5. By increasing our knowledge about web design, we were able to create noticable different projects from freshmen year.

We also learned about github and learned a new language and a new way to keep track of your branches within a website.
